# AWS Static Website Deployment 🚀

A static website deployed on Amazon Web Services (AWS) as part of the Udacity/Accenture Cloud Learning Module.

## 🌐 Live Website
- **CloudFront URL:** https://d17f86g2mgig6h.cloudfront.net
- **S3 URL:** http://my-website-2025-udita.s3-website-us-east-1.amazonaws.com

## 🛠️ Technologies Used
- HTML5, CSS3, JavaScript
- Amazon S3 (Storage & Hosting)
- Amazon CloudFront (CDN)
- AWS IAM (Bucket Policy)

## 📋 Steps Followed
1. Created S3 bucket
2. Enabled static website hosting
3. Added IAM bucket policy for public access
4. Uploaded website files
5. Created CloudFront distribution

## 📁 Project Files
- index.html - Main webpage
- style.css - Styling
- script.js - JavaScript functionality
